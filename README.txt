Indexador e Buscador de Palavras
Programa desenvolvido na linguagem C utilizando técnicas de Algoritmos e Estrutura de Dados 1.

Este programa busca palavras em um arquivo de texto, o programa retornará o número de ocorrências de determinada palavra em um texto (arquivo txt), exibindo a linha de cada ocorrência e o tempo de execução para cada procura e o tempo para criar a estrutura escolhida (lista ou arvore). Para executar nosso programa recomendamos que seja pelo PowerShell.
Basta caminhar até o diretório do arquivo, inserir gcc -o program program.c e dar enter. Após isso, inserir ./program texto.txt arvore ou ./program texto.txt lista
